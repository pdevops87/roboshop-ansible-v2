import_tasks:
=============
to call a yaml file into another within same folder , this is static parse

ex:
--
import_tasks:
   file: fil1.yaml

include_tasks:
==============
to call a yaml file into another within same folder dynamically

ex:
---

include_tasks:
   file: {{ file }}.yaml


role wise:
==========
to call a role name (under role: every folder is a role name) into another folder

- name: print a message
  hosts: localhost
  connection: local
  tasks:
